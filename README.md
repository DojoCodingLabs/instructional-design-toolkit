# instructional-design-toolkit

Plugin de Claude Code para diseño instructional guiado — cursos completos cmi5-compliant
y planes de sesiones 1-on-1 (coaching / mentoring / tutoring).

Status: en desarrollo. Ver `docs/superpowers/specs/2026-04-17-instructional-design-toolkit-design.md`
y `docs/superpowers/plans/2026-04-17-instructional-design-toolkit-plan.md` en el repo
parent `dojocoding/`.
